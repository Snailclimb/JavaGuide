## Java

- [JavaGuide](https://github.com/Snailclimb/JavaGuide "JavaGuide"): [Java Learning + Interview Guide] A guide that covers the core knowledge that most Java programmers need to master.
- [toBeBetterJavaer](https://github.com/itwanger/toBeBetterJavaer): An easy-to-understand, humorous Java learning guide, covering Java basics, Java collection framework, Java concurrent programming, JVM, Java enterprise development (Git, SSM, Spring Boot) and other knowledge points.
- [interview-guide](https://github.com/csguide-dabai/interview-guide): Summarizes the key points in the eight-part article on back-end interviews, hoping to help you prepare for school recruitment interviews for Internet development positions.
- [advanced-java](https://github.com/doocs/advanced-java "advanced-java"): Internet Java engineers are completely literate in advanced knowledge: covering high concurrency, distribution, high availability, microservices, massive data processing and other fields.
- [toBeTopJavaer](https://github.com/hollischuang/toBeTopJavaer "toBeTopJavaer"): The road to becoming a Java engineer.
- [technology-talk](https://github.com/aalansehaiyang/technology-talk): Summarizes common technical frameworks in the Java ecosystem, open source middleware, system architecture, databases, large company architecture cases, commonly used third-party libraries, project management, online problem troubleshooting, personal growth, thinking and other knowledge
- [JCSprout](https://github.com/crossoverJie/JCSprout): A core Java knowledge base in its infancy.
- [bestJavaer](https://github.com/crisxuan/bestJavaer): This is a series of tutorials to become a better Java programmer.
- [java-design-patterns](https://github.com/iluwatar/java-design-patterns "java-design-patterns"): Design patterns implemented in Java.

## Computer Basics

- [cs-self-learning](https://github.com/PKUFlyingPig/cs-self-learning): Computer self-study guide, summarizing high-quality computer courses from many famous European and American schools.

## System Design

### SpringBoot

- [springboot-guide](https://github.com/Snailclimb/springboot-guide): Summary of SpringBoot core knowledge points. Based on Spring Boot 2.19+.
- [SpringAll](https://github.com/wuyouzhuguli/SpringAll "SpringAll"): Step by step, learn Spring Boot, Spring Boot & Shiro, Spring Cloud, Spring Security & Spring Security OAuth2, blog Spring series source code.
- [Springboot-Notebook](https://github.com/chengxy-nds/Springboot-Notebook): A series of development frameworks based on Spring Boot, integrating mainstream Internet technologies such as Redis, Rabbitmq, ES, MongoDB, Spring Cloud, Kafka, Skywalking, etc., to implement comprehensive cases of various common function points.
- [springboot-learning-example](https://github.com/JeffLi1993/springboot-learning-example "springboot-learning-example"): Spring Boot practical learning case, which is the best practice for Spring Boot beginners and core technology consolidation.
- [spring-boot-demo](https://github.com/xkcoding/spring-boot-demo "spring-boot-demo"): spring boot demo is a project for in-depth learning and practical use of spring boot. Currently, it contains a total of 63 integrated demos, 52 of which have been completed.
- [SpringBoot-Labs](https://github.com/YunaiV/SpringBoot-Labs): Spring Boot series tutorials.

## Open source books

- ["Philosophical Principles of High Concurrency"](https://github.com/johnlui/PPHC): The goal of this book is to discuss a philosophical principle hidden behind high concurrency problems within the author's limited cognitive scope-to find a single point and split it.
- ["Effective Java (3rd Edition)" Chinese-English bilingual version](https://github.com/clxering/Effective-Java-3rd-edition-Chinese-English-bilingual): Chinese and English learning reference for each chapter of "Effective Java (3rd Edition)".- ["DDIA (Designing Data-Intensive Application)" Chinese version](https://github.com/Vonng/ddia): "Designing Data-Intensive Application" DDIA Chinese translation.
- ["Phoenix Architecture"](https://github.com/fenixsoft/awesome-fenix): Discuss how to build a reliable large-scale distributed system.
- ["Chinese version of "Patterns of Distributed Systems"](https://github.com/dreamhead/patterns-of-distributed-systems): Chinese translation of "Patterns of Distributed Systems".
